[out]omaxAchieves the same result as:

BoundsClear(omins, omaxs); for each corner c in bounds{imins, imaxs} { vec3_t p; MatrixTransformPoint(m, c, p); AddPointToBounds(p, omins, omaxs); }

With fewer operations

Pseudocode: omins = min(mins.x*m.c1, maxs.x*m.c1) + min(mins.y*m.c2, maxs.y*m.c2) + min(mins.z*m.c3, maxs.z*m.c3) + c4 omaxs = max(mins.x*m.c1, maxs.x*m.c1) + max(mins.y*m.c2, maxs.y*m.c2) + max(mins.z*m.c3, maxs.z*m.c3) + c4

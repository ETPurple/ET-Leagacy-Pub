void HT_Apply ( hashtable_t  table,
ht_apply_funct  function,
void *  data 
)

Applies a function to all items in the table.

Parameters
[in]tableThe hash table onto which the function is to be applied
[in]functionPointer to the function to apply
[in]dataExtra data to pass as the function's second parameter
Note
The order in which the function is applied is either the insertion order or, if the table has HT_FLAG_SORTED set, the increasing key order.
The function should return false if processing is to stop.

Detailed Description

This file does all of the processing necessary to turn a raw grid of points read from the map file into a srfGridMesh_t ready for rendering.

The level of detail solution is direction independent, based only on subdivided distance from the true curve.

Only a single entry point: R_SubdividePatchToGrid(int width, int height, srfVert_t points[MAX_PATCH_SIZE*MAX_PATCH_SIZE])
